# The GitOps Dictionary

*Universal reference for GitHub Projects V2 Metadata.*

## 1. Field Types

| Field | Type | Description |
|-------|------|-------------|
| **Status** | Single Select | Todo, In Progress, Done |
| **Priority** | Single Select | Critical, High, Medium, Low |
| **Type** | Single Select | Epic, Story, Task, Bug |
| **Size** | Number | Story Points (Fibonacci) |

## 2. Standard Labels

*   `priority:critical` (#D4380D)
*   `priority:high` (#FA8C16)
*   `type:bug` (#d73a4a)
*   `type:feature` (#a2eeef)
